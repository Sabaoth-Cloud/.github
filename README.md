# GitHub Templates

This directory contains issue templates and pull request templates for the repository.

## Structure

```
.github/
├── ISSUE_TEMPLATE/
│   ├── bug_report.yml        # Bug report template
│   ├── feature_request.yml   # Feature request template
│   └── config.yml            # Issue template configuration
├── PULL_REQUEST_TEMPLATE.md  # Pull request template
└── README.md                 # This file
```

## Usage

### Creating issues

1. Click on "New issue" in the GitHub repository
2. Choose from the available templates:
   - Bug report - for reporting bugs
   - Feature request - for suggesting new features
3. Fill out the template with relevant details

### Creating pull requests

1. Push your changes to a branch
2. Open a pull request
3. The PR template will be automatically loaded
